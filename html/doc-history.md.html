KMS encryption support for training and hosting Amazon SageMaker now supports KMS encryption for hosting instances and training model artifacts at rest. You can specify a AWS Key Management Service key that Amazon SageMaker uses to encrypt data on the storage volume attached to a hosting endpoint by using the KmsKeyId request parameter in a call to CreateEndpointConfig. You can specify an AWS KMS key that Amazon SageMaker uses to encrypt training model artifacts at rest by setting the KmsKeyId field of the OutputDataConfig object you use to configure your training job.
